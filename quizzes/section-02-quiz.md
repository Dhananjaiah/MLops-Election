### Question 2

What is the purpose of a virtual environment in Python?

- A) To make code run faster
- B) To isolate project dependencies
- C) To compile Python code
- D) To connect to databases

<details>
<summary>Answer</summary>

**B** - Virtual environments isolate dependencies between projects.

</details>

### Question 3

You need a powerful GPU to learn MLOps concepts.

- A) True
- B) False

<details>
<summary>Answer</summary>

**False** - Most MLOps concepts can be learned on a regular laptop.

</details>

### Question 4

Which command creates a new Git branch?

- A) git merge
- B) git checkout -b
- C) git push
- D) git status

<details>
<summary>Answer</summary>

**B** - git checkout -b creates and switches to a new branch.

</details>

### Question 5 (Short Answer)

List three common setup issues you might encounter and how to resolve them.

<details>
<summary>Sample Answer</summary>

1) Port conflicts - change the port or stop conflicting services. 2) Docker permissions - add user to docker group. 3) Corporate proxy - configure proxy settings in pip/docker.

</details>
