# Assets - Images

This directory contains images used throughout the course.

## Categories

### Screenshots
- Tool interfaces (MLflow, Docker, Kubernetes, etc.)
- Terminal outputs
- API responses

### Icons
- Technology logos
- Role/team icons
- Status indicators

### Photos
- (Optional) Instructor photos
- Team collaboration images

## Image Requirements

- **Format**: PNG for screenshots, SVG for icons
- **Quality**: High resolution (retina-ready)
- **Size**: Optimized for web (<500KB)
- **Alt text**: Include in Markdown for accessibility

## Usage

Reference images in lecture files:
```markdown
![Description](../../assets/images/image-name.png)
```
